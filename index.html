The Isnad Chain

A privacy-preserving skill attestation registry for AI agents. Auditors verify skills anonymously using ZK proofs. Trust scores are public. Auditor identities are not.

⬡
Anonymous Attestation
Auditors submit ZK proofs. Their identity never touches the chain — only the trust signal does.
⬡
Content-Addressed
Trust is tied to SHA256 of skill content. Modify the file, the hash changes, prior attestations expire.
⬡
Private Credential Vault
Store API keys as private notes on Aztec L2. Grant scoped access per skill via AuthWit.

How it works

⬡

Anonymous attestation

Auditors examine skill files and submit ZK attestations. Their identities never appear in public state — only the trust score increments.

⬡

Content-addressed

Trust scores are keyed to the SHA256 of skill file content. If the file changes, the hash changes — prior attestations no longer apply.

⬡

Private credential vault

Store API keys and secrets as private notes on Aztec. Grant scoped access to individual skills via AuthWit — never expose your full vault.
